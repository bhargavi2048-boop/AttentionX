01

Core AI Engine — Multimodal & Gen AI APIs

You need an API that can understand video or text transcripts at scale.

✦

Google Gemini 1.5 Flash

Multimodal AI

Massive 1M+ token context window — you can upload an entire long video or its transcript for analysis. Excellent at finding "the best parts" and generating hook headlines.

Google AI Studio ↗
◆

OpenAI Whisper

Speech-to-Text

The industry standard for converting video audio into text with high-accuracy word-level timestamps. Powers the karaoke-style caption generation pipeline.

OpenAI Speech-to-Text ↗
02

Video & Audio Processing

Python libraries for the heavy lifting of cutting, cropping, and analyzing media.

🎬

MoviePy

Video Processing

For cutting, cropping, compositing, and exporting video clips. Handles all video editing operations programmatically with frame-level precision.

MoviePy Documentation ↗
🔊

Librosa

Audio Analysis

For analyzing audio "loudness" to find where the mentor is speaking most passionately. RMS energy tracking maps emotional peaks across the timeline.

Librosa Guide ↗
👁

MediaPipe

Computer Vision

For Face Detection to help with smart-cropping to vertical. Google's ML framework provides real-time face landmark detection at 30fps+ for smooth tracking.

MediaPipe Solutions ↗

Backend — Python-Based API

FastAPI

API Framework

High performance and very easy to document. Async support handles long video processing tasks without blocking. Auto-generated OpenAPI docs included.

Flask

Lightweight Framework

Simple and great for quick hackathon prototypes. Minimal boilerplate, instant routing, and easy to deploy for rapid iteration during development.
